DEXT5 Editor :: Compatibility

XML : <dingbat_char>.....</dingbat_char>, Config : config.Compatibility.DingBatChar

Config 설정이 우선 적용됩니다.
붙여넣기 및 SetAPI 사용 시 호환성 항목을 설정합니다.

remarks

XML : dingbat_char, Script : Compatibility.DingBatChar  DingBat Font를 특수문자로 변환합니다.
기본값은 "0"이고, 특수문자 변환을 하지 않고, "1"로 설정 시 붙여넣기 및 SetAPI 동작 시 변환합니다
"paste" 설정 시 붙여넣기 동작일 때만 변환하고, "set"으로 설정 시 SetAPI 동작일 때만 변환합니다.
config/dext_dingbat_font.js 파일에 변환할 폰트 리스트와, 대체 할 문자 등을 정의할 수 있습니다.

sample code

paste,setAPI 일때 DingBat font를 특수 문자로 변환합니다. 

XML

.....
<setting>
    .....
    <compatibility>
        .....
        <dingbat_char>1</dingbat_char>
        .....
    </compatibility>
    .....
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>
