DEXT5 Editor :: dext_editor_apply_font_style_event

void dext_editor_apply_font_style_event(editorID, fontStyleObject)

폰트 스타일(fontFamily, fontSize, lineHeight) 적용 시 발생합니다.

return value

없음.

parameters

editorID  이벤트가 발생한 에디터의 id입니다.
fontStyleObject  적용된 폰트 스타일(fontFamily, fontSize, lineHeight) 정보를 포함한 객체입니다.

remarks

fontStyleObject는 아래의 정보를 담고 있습니다.
 - fontStyleObject.style = 적용된 폰트 스타일 이름
 - fontStyleObject.value = 적용된 폰트 스타일 값
